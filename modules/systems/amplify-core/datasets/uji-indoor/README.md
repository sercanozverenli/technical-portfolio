# UJIIndoorLoc Dataset

This project uses the official **UJIIndoorLoc** indoor positioning dataset.

The dataset is **not included** in this repository because it is publicly available and consists of large static files.

## Download

Download the official dataset and place the following files into the `raw/` directory.

Expected structure:

```text
raw/
├── trainingData.csv
└── validationData.csv
```

The original files should not be modified before preprocessing.
